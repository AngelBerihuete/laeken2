Synthetic dataset generated from real Austrian EU-SILC.

Usage

data(eusilc2)

Format

A data frame with 6000 rows of 7 variables:

DB010
integer; year of the pull

DB020
factor; name of the country

DB040
factor; name of the region within the country

Description

The dataset is the same as in laeken package but with transformed variables in order to perform right calculations using laeken2.

References

Moore et al. (2013) Genetics 195:1077-1086 (PubMed)
