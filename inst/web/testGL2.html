Test for Lorenz and Generalized Lorenz (GL) dominance

Usage

testGL2(dataset1, dataset2, generalized = FALSE, samplesize = 10)

Arguments

generalized
logical; if TRUE the test will be applied to two Generalized Lorenz curves.
samplesize
an integer which represents the number of Lorenz (GL) curve ordinates to be estimated.
